Ошибки метода isTeenager()
Мной были использованы следующие тестовые данные: 
  10 
  12
  13
  15
  19
  20
  25
Тест показал следующие результаты:
  Run 1: PersonTest.testisTeenager:25 expected:<true> but was:<false>
  Run 2: PersonTest.testisTeenager:25 expected:<true> but was:<false>
  Run 3: PASS
  Run 4: PASS
  Run 5: PersonTest.testisTeenager:25 expected:<false> but was:<true>
  Run 6: PASS
  Run 7: PASS
На основании данных результатов мы можем выявить следующие проблемы метода:
  По заданию возраст тинейджеров должен быть в диапазоне между 13 и 19 включительно,
  однако, верхняя граница метода выглядит как (age < 19), что является ошибкой, так как 19 должно включаться в правильный диапазон,
  а возвращает false.
  Так же минмальный возраст должен быть 13, однако результаты теста показывают, что 10 и 12 тоже включены методом в категорию тинейджеров
  (возвращают true), так как нижняя граница не обозначена.
